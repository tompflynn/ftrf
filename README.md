FTRF, a new language that compiles to the r-word language

## Usage

1. clone this repository
2. run these commands
```
cd ftrf
chmod +x ftrfcompiler.sh
cp ftrfcompiler.sh /usr/bin/
```
3. write FTRF code and run the compiler
